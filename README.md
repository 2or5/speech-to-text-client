## General info
This project was created so that the user can create notes with the help of voice. The user can record his voice, send it to the back, end and receive his recorded voice as text. If necessary, the user can edit and delete notes. Also, you can visit speech-to-text (https://github.com/2or5/Speech-to-text).

## Features

- Interface for uploading audio files.
- View processing status of transcriptions.
- Login and Registration forms.
- View history of transcribed notes.

## Technologies

- **React.js**
- **JavaScript / TypeScript**
- **Axios** (for API requests)
- **CSS**
- **NPM**

## Setup
To run this project, install it locally :

### 1. Clone the repository

Open your terminal and run:
```bash
git clone [https://github.com/nazarbokalo/Voice-Transcriber-Frond-End.git](https://github.com/nazarbokalo/Voice-Transcriber-Frond-End.git)
cd Voice-Transcriber-Frond-End
```

### 2. Install dependencies

```bash
npm install
```
### 3. Run the application
```bash
npm start
```
The Frontend will start at: http://localhost:3000
